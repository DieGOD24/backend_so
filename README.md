## Requisitos
- Python 3.12
- Django

## Instalación
```bash
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
- Planificador: http://127.0.0.1:8000/sim/
- VFS: http://127.0.0.1:8000/vfs/
